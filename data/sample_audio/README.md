# Sample Audio Files

Test audio recordings for VoiceBridge experiments.

## Recording Your Own

Use QuickTime Player on Mac:
1. File > New Audio Recording
2. Select built-in microphone
3. Record test phrases
4. Save as .wav format

## Test Phrases Used

- "Hello, how are you?"
- "Where is the nearest hospital?"
- "I need help with my computer."
- "What time does the store close?"
- "Thank you for your assistance."

Recordings saved as test_1.wav, test_2.wav, etc.
